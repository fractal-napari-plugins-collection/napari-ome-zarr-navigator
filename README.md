A plugin to browse OME-Zarr plates by conditions and load images, labels and features from ROIs


## Usage

The Plate Browser supports loading a plate from the local filesystem or via authenticatd https calls. Additionally, the Plate Browser recognizes OME-Zarr plates that were loaded via [napari-ome-zarr](https://github.com/ome/napari-ome-zarr). It allows browsing the selection of wells in the multiwell plate.

[screenshot init]

The Plate Browser allows to zoom to a given well ("Go to well") & puts a white bounding box around the selected well.

[screenshot go-to-well]

Using the ngio `condition tables` (see [ngio condition table definition](https://biovisioncenter.github.io/ngio/stable/table_specs/table_types/condition_table/)), the Plate Browser allows for selecting subsets of the well list based on combinations of conditions.

[screenshot condition table]

The ROI loader can be used standalone or integrated with the Plate Browser. If a well is selected from the Plate Browser, all the images in that well can be loaded via the ROI loader.
This supports:
- Loading images from different multiplexing acquisitions
- Loading any ROI based on [Fractal ROI tables](TODO new links)
- Loading whole images without ROI tables
- Loading label images
- Loading feature measurements (based on [feature tables](https://biovisioncenter.github.io/ngio/stable/table_specs/table_types/feature_table/))

The ROI loader supports both loading images and labels as a lazy, multi-resolution pyramid (optimized for streaming) and at fixed resolutions (optimized for interactivity e.g. for 3D images and for downstream usage in tools like the [napari-feature-classifier](https://github.com/fractal-napari-plugins-collection/napari-feature-classifier))

This approach of loading label images and feature data has been optimized for and tested with the [napari feature classifier](https://github.com/fractal-napari-plugins-collection/napari-feature-classifier).

[screenshot ROI loader]

This plugin is tested to work well for OME-Zarr files generated by [Fractal](https://fractal-analytics-platform.github.io/) and meant to work with other OME-Zarrs (of at least v0.4).
